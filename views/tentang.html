Pymin adalah sebuah sistem manajemen Virtual Private Server yang bebasiskan Vserver. Pymin dibangun dengan menggunakan bahasa Python.

Untuk dapat menjalankan Pymin anda membutuhkan beberapa aplikasi pendukung sebagai berikut:

  1. Python 2.4 atau lebih tinggi
  2. Slackware 13.1 Gnu/linux
  3. Patch kernel vs2.3.0.36.30.4
  4. Kernel 2.6.33.7
  5. Util-Vserver 0.30.215
Cara membangun Server Virtual Private Server barbasis Vserver dapat anda lihat pada http://research.bramandityo.com/
